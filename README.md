# MyHttp
## Overview
MyHttp is a HTTP server created for learning purposes, it supports GET and POST requests.
## Installation
### Requirements
- Linux-Based OS
- C Compiler
### Build
```
# Clone the repository
https://github.com/Smet-k/MyHttp
cd MyHttp

# Build the utility
./build.sh

# Run
sudo ./build/MyHttp
```
## Usage
```
MyHttp
```
Configuration is done through `config.cfg`
```cfg
port=8080
threads=8
```
